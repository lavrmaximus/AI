```markdown
Этот файл представляет собой код для интеллектуального диалогового менеджера, предназначенного для сбора данных о бизнесе в свободной форме. Основные компоненты кода включают:

1. **Классы**:
   - `BusinessConversation`: управляет сессией диалога, состояниями (начало, ожидание имени бизнеса, сбор данных, готовность к анализу и завершение), и обеспечивает обработку пользовательских сообщений.
   - `ConversationManager`: управляет активными сессиями, позволяя создавать или завершать диалоги для пользователей.

2. **Состояния**: Диалог проходит через несколько состояний, таких как "START", "AWAITING_BUSINESS_NAME", "COLLECTING_DATA" и другие.

3. **Сбор данных**: Менеджер собирает необходимые и дополнительные данные о бизнесе, обрабатывает их с помощью AI, анализирует недостающие данные и подводит итоги перед анализом.

4. **Анализ данных**: После сбора всех необходимых данных, запускается анализ, результаты которого форматируются и возвращаются пользователю.

5. **Логирование и обработка ошибок**: Код содержит механизмы для логирования действий и обработки возможных ошибок.

6. **AI-компоненты**: Используются внешние модули для извлечения и анализа данных, а также для форматирования отчетов.

Таким образом, код представляет собой комплексное решение для сбора и анализа информации о бизнесе с использованием диалогового интерфейса.
```
